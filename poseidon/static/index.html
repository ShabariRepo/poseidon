Session memory

Everything in this chat — your messages, Poseidon's replies, and what its tools returned — rides along on every turn so the model remembers the conversation. That working memory (the context) has a size limit.

When the bar fills, Poseidon automatically summarizes the older part of the session into a dense brief and keeps the recent messages verbatim, so the conversation just keeps going. Nothing is deleted — the full history stays saved, and durable facts live in project Memory.

The limit is tunable in Settings → Engine (lower it for small-context models).
